Predefined SAPUI5 CSS Class

A reference list of predefined SAPUI5 CSS classes, including margin, padding, content density, and other utility classes. Double-click a class name to add it to the styleClass property.

Full Margins

Clears space on all sides of the control.

Class Description
sapUiTinyMargin 8px margin on all four sides.
sapUiSmallMargin 16px margin on all four sides.
sapUiMediumMargin 32px margin on all four sides.
sapUiLargeMargin 48px margin on all four sides.

Single-Sided Margins

Clears space on a single side (Top / Bottom / Begin / End). Begin is left in LTR and right in RTL.

Class Description
sapUiTinyMarginTop 8px margin on the top side.
sapUiTinyMarginBottom 8px margin on the bottom side.
sapUiTinyMarginBegin 8px margin on the begin side (left in LTR, right in RTL).
sapUiTinyMarginEnd 8px margin on the end side (right in LTR, left in RTL).
sapUiSmallMarginTop 16px margin on the top side.
sapUiSmallMarginBottom 16px margin on the bottom side.
sapUiSmallMarginBegin 16px margin on the begin side (left in LTR, right in RTL).
sapUiSmallMarginEnd 16px margin on the end side (right in LTR, left in RTL).
sapUiMediumMarginTop 32px margin on the top side.
sapUiMediumMarginBottom 32px margin on the bottom side.
sapUiMediumMarginBegin 32px margin on the begin side (left in LTR, right in RTL).
sapUiMediumMarginEnd 32px margin on the end side (right in LTR, left in RTL).
sapUiLargeMarginTop 48px margin on the top side.
sapUiLargeMarginBottom 48px margin on the bottom side.
sapUiLargeMarginBegin 48px margin on the begin side (left in LTR, right in RTL).
sapUiLargeMarginEnd 48px margin on the end side (right in LTR, left in RTL).

Two-Sided Margins

Clears space on two opposite sides at once (BeginEnd = horizontal, TopBottom = vertical).

Class Description
sapUiTinyMarginBeginEnd 8px margin on both begin and end sides (horizontal).
sapUiTinyMarginTopBottom 8px margin on both top and bottom sides (vertical).
sapUiSmallMarginBeginEnd 16px margin on both begin and end sides (horizontal).
sapUiSmallMarginTopBottom 16px margin on both top and bottom sides (vertical).
sapUiMediumMarginBeginEnd 32px margin on both begin and end sides (horizontal).
sapUiMediumMarginTopBottom 32px margin on both top and bottom sides (vertical).
sapUiLargeMarginBeginEnd 48px margin on both begin and end sides (horizontal).
sapUiLargeMarginTopBottom 48px margin on both top and bottom sides (vertical).

Negative Margins

Applies a two-sided (BeginEnd) negative margin. Useful for aligning elements with built-in paddings.

Class Description
sapUiTinyNegativeMarginBeginEnd -8px negative margin on both begin and end sides.
sapUiSmallNegativeMarginBeginEnd -16px negative margin on both begin and end sides.
sapUiMediumNegativeMarginBeginEnd -32px negative margin on both begin and end sides.
sapUiLargeNegativeMarginBeginEnd -48px negative margin on both begin and end sides.

Responsive Margin

Adapts its margin based on the available screen width using media queries.

Class Description
sapUiResponsiveMargin Screen < 600px: 16px bottom only. 600–1023px: 16px on all sides. > 1023px: 16px top/bottom and 32px begin/end.

Content Density

Switches the overall size and spacing of controls. Set on a parent element (commonly

Class Description
sapUiSizeCozy Cozy density — larger dimensions and spacing optimized for touch-enabled devices. Default density for most sap.m controls.
sapUiSizeCompact Compact density — reduced control dimensions and spacing (font size unchanged), optimized for desktop and non-touch devices operated by mouse.

Utility

Helper classes that complement the margin classes.

Class Description
sapUiForceWidthAuto Overrides a control’s default 100% width with auto. Useful for controls without a width property (e.g. sap.m.IconTabBar).

No Margin

Removes margins from controls that ship with default margins.

Class Description
sapUiNoMargin Removes margins on all four sides.
sapUiNoMarginTop Removes the top-side margin.
sapUiNoMarginBottom Removes the bottom-side margin.
sapUiNoMarginBegin Removes the begin-side margin (left in LTR, right in RTL).
sapUiNoMarginEnd Removes the end-side margin (right in LTR, left in RTL).
