이미지 압축(Compress) 옵션 가이드

파일 업로드 시 이미지를 자동으로 압축하여 용량을 줄이는 기능에 대한 가이드입니다.

기능 설명

이미지 압축 옵션을 활성화하면, 사용자가 파일을 업로드하는 시점에 브라우저에서 이미지를 자동으로 압축합니다. 압축이 완료된 파일이 서버로 전달되므로 업로드 속도가 빨라지고 저장 공간을 절약할 수 있습니다.

옵션 비활성화 시

압축 없이 원본 이미지 파일이 그대로 서버에 전달됩니다. 파일 크기가 클 경우 업로드 속도가 느려질 수 있습니다.

옵션 활성화 시

파일 업로드 시점에 브라우저에서 이미지를 압축한 후 서버로 전달합니다. 이미지 화질은 아래 이미지 품질(Quality) 설정값에 따라 조절됩니다.

처리 흐름

단계 설명
파일 선택 사용자가 이미지 파일 선택
파일 형식 확인 jpg · png · webp 여부 판단
브라우저 압축 품질 설정값 기준으로 압축
서버 전송 압축된 파일 업로드

⚠️ 지원되지 않는 파일 형식(GIF, BMP, SVG 등)은 압축 없이 원본 그대로 서버에 전달됩니다.

압축 지원 이미지 형식

아래 세 가지 형식의 이미지 파일에 한해 압축이 적용됩니다. 이 외의 파일은 압축 없이 원본이 업로드됩니다.

  • .jpg / .jpeg — 일반 사진 파일에 적합
  • .png — 투명 배경 이미지 지원
  • .webp — 차세대 고효율 이미지 형식

적용 예시

항목 수치
원본 파일 크기 5 MB
압축 후 파일 크기 1.5 MB (▼ 70% 절감, 품질 0.3 기준)

ℹ️ 위 수치는 예시입니다. 실제 압축 결과는 이미지 종류·해상도·품질 설정값에 따라 달라질 수 있습니다.

이용 안내

  1. 옵션 활성화 하단 설정 영역에서 이미지 압축 옵션을 ON으로 설정합니다.

  2. 이미지 품질 설정 품질 값은 0.01 ~ 1 사이의 숫자로 설정합니다. 숫자가 낮을수록 파일 크기가 작아지고, 높을수록 원본에 가까운 화질이 유지됩니다.

  3. 파일 업로드 이미지 파일을 업로드하면 브라우저에서 자동으로 압축이 진행됩니다. 별도의 작업 없이 압축된 파일이 서버에 저장됩니다.

  4. 결과 확인 업로드 완료 후 원본 파일 크기와 압축 후 파일 크기를 화면에서 확인할 수 있습니다.
