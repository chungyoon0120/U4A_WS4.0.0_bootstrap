Shortcut Item List Screen – User Guide

This document explains how to review registered shortcut keys on the "Shortcut Event Where to Use" screen and how to navigate to the related UI event locations and server event (ABAP source).

1. Screen Overview

The Shortcut Item List screen displays all shortcut keys registered in the application in a list format. From this screen, you can quickly navigate to the target UI event location and the corresponding server event ABAP source.

Link Behavior
  • Clicking the event name in Event Target Properties (for example, press) moves to the target UI event location.
  • Clicking the Event ID (for example, EV_TEST) automatically launches SAP GUI and navigates to the ABAP source of the corresponding server event.

1.1 Column Description

Column Description
Hotkey Displays the registered shortcut key combination (e.g. Ctrl+D, Ctrl+Shift+Q).
Event Target Properties Displays the UI event name linked to the shortcut. Clicking it navigates to the UI event definition.
Event ID Displays the server event ID. Clicking it opens SAP GUI and navigates to the ABAP source.
UI OBJ ID Displays the object ID of the UI where the shortcut is registered (e.g. BUTTON2, BUTTON3).
Event Text Displays the descriptive text or label of the event (e.g. TEST).

1.2 Search

Use the search field at the top of the screen to quickly filter the list by Hotkey, event name, Event ID, or UI Object ID.

1.3 Refresh

Clicking the Refresh button (⟳) in the upper-right corner reloads the list and retrieves the latest shortcut key information currently applied to the application.
